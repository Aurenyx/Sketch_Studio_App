
# 🎨 Kids Drawing App - Shape Imagination 

Shape Imagination is a fun, creative, and kid-friendly drawing app designed to help children explore their imagination through colors, shapes, and free drawing.
The interface is clean, colorful, and super easy for kids to understand — all tools are visual and instantly accessible.


## ✨ Features

### 1.🖍 Drawing Tools

-Smooth & responsive free-hand drawing

-Multiple brush colors (vivid & soft shades)

-Adjustable brush stroke

-Soft round brush for a natural drawing feel


### 2.🎨 Colors & Palette

-10 kid-friendly colors (yellow, green, orange, red, blue, purple, black, white, etc.)

-Easy tap-to-select color buttons

-Large color boxes for kids’ convenience


### 3.🖼 Canvas Editing

-Flexible drawing canvas inside a stylish frame

-Undo & redo buttons for easy corrections

-Clean and minimal drawing space


### 4.📷 Image Import

-Import images from the gallery

-Draw on top of photos

-Useful for doodling, tracing shapes, or decorating pictures


### 5.🧒 Kid-Friendly UI

-Big buttons

-Cartoon-style icons

-Intuitive layout

-Designed for children aged 3–12
